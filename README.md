🧠 Sobre o Projeto

Este projeto consiste em dois principais fluxos de usuário (Flow 1 e Flow 2) que representam a navegação e interação do usuário com o app:

🔹 Flow 1

Um primeiro fluxo simplificado para iniciar a experiência do usuário — geralmente relacionado ao login e integração com outro app (como WhatsApp), para autenticação rápida.

Tela de login

Navegação inicial

Interação com menus externos

🔹 Esse fluxo mapeia os primeiros passos do usuário ao abrir o app pela primeira vez.

🔹 Flow 2

Fluxo principal com múltiplas telas e interações:

✔ Tela de login e autenticação
✔ Tela principal com Cursos/Disciplinas
✔ Agenda de atividades
✔ Notificações
✔ Detalhes de curso
✔ Navegação entre telas via botões e ícones

Esse fluxo detalha como o usuário navega entre seções do app — incluindo como acessar listas de disciplinas, visualizar atividades, consultar agenda e ver detalhes específicos de cada curso.
